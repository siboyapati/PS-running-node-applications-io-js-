To run this examples:
- Open the Terminal for the application (for example e6-demo)
- Write “npm install” to install all the dependencies

Note
For the koa-demo you need MongoDb installed and running: http://www.mongodb.org/

Run tests
- In the application folder write “npm test” in the terminal

Start the applications
- Write “npm start” in the terminal

For the es6-demos you will have to start each file separately “iojs shipping.js”, and then use the flags shown in the course
